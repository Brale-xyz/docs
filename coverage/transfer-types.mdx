title: 'Transfer Types'
description: 'Supported transfer_type identifiers and names for moving value on/off chain.'
---

Brale supports these transfer types for funding and payouts. Use them in `transfer_type` fields on addresses and transfers. Source: [TransferType.json](https://github.com/Brale-xyz/commons/blob/main/Resources/TransferType.json).

## Onchain

| transfer_type (API id) | Name       |
| ---------------------- | ---------- |
| `ethereum`             | Ethereum   |
| `base`                 | Base       |
| `solana`               | Solana     |
| `canton`               | Canton     |
| `stellar`              | Stellar    |
| `polygon`              | Polygon    |
| `avalanche`            | Avalanche  |
| `optimism`             | Optimism   |
| `arbitrum`             | Arbitrum   |
| `hedera`               | Hedera     |
| `celo`                 | Celo       |
| `kusama`               | Kusama     |
| `spark`                | Spark      |
| `xrp_ledger`           | XRP Ledger |

## Offchain

| transfer_type (API id)    | Name                  |
| ------------------------- | --------------------- |
| `wire`                    | Wire Transfer         |
| `ach-debit`               | ACH Debit             |
| `ach-credit`              | ACH Credit            |
| `same-day-ach-credit`     | Same Day ACH Credit   |
| `rtp-credit`              | RTP Credit            |

## Testnet transfer types

Use these values when working in testnet. They mirror the corresponding mainnet chains.

| transfer_type (API id) | Network                |
| ---------------------- | ---------------------- |
| `base_sepolia`         | Base Sepolia Testnet   |
| `canton_testnet`       | Canton Testnet         |
| `amoy`                 | Polygon Amoy Testnet   |
| `sepolia`              | Ethereum Sepolia Testnet |
| `solana_devnet`        | Solana Devnet          |
| `xion_testnet`         | Xion Testnet           |
| `tempo_testnet`        | Tempo Testnet          |
