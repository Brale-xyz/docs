title: "Documentation MCP"
---

Our Documentation MCP Server exposes our developer documentation directly into your MCP-Compatible LLM client.  Query our developer documentation directly in your workflow, and write code with lastest information.

Most LLMs have unique terminology and steps to add an MCP server, yet they all require a display name and the remote server URL.

```
name:  Brale.xyz Documentation MCP
Remote MCP Server URL: https://docs.brale.xyz/mcp
```
