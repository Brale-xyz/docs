---
title: "Canton Token transfers to EOAs"
---

The Canton network is rapidly evolving, and they recently launched support for one-step token transfers.

In order to receive tokens to your EOA, you must disable the auto-accept utility token transfer features that some wallets are quickly rolling out.

We are currently building out support for 1-step token transfers, so this requirement is temporary.
