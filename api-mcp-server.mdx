---
title: "API MCP"
---

The Brale API MCP Server allows you to interact with our API directly with your MCP-Compatible LLM client.  You can perform operations directly against our API, without writing a line of code.

There are three ways to leverage the Brale API MCP server, depending on your needs and LLM of choice.

1. Hosted Server with Oauth2 credentials
2. Hosted Server with Custom Headers
3. Self-Hosted Docker Image
